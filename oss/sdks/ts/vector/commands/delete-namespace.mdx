---
title: Delete Namespace
---

Used to delete namespace of an index.

## Arguments

<ResponseField name="namespace" type="string" required>
  The name of the namespace to delete.
</ResponseField>

## Response

This method returns `'OK'` if the namespace is successfully deleted.

<RequestExample>

```typescript Basic
const responseDelete = await index.deleteNamespace("namespace-example");
//"OK"
```

</RequestExample>
