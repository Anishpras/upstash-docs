---
title: Reset
---

Used to delete every vector in an index.

## Arguments

<ResponseField name="options" type="Object">
  <Expandable>
    <ResponseField name="namespace" type="string">
      The namespace to use. Leave empty to use the default namespace. For more
      information on namespaces see the [dedicated
      section](/vector/features/namespaces).
    </ResponseField>
  </Expandable>
</ResponseField>

## Response

`'Success'` if the index is successfully resetted.

<RequestExample>

```typescript Basic
const responseReset = await index.reset();
// 'Successful'
```

</RequestExample>
