---
title: List Namespaces
---

Used to retrieve the list of namespaces in an index.

## Arguments

This method doesn't require any additional data.

## Response

<ResponseField name="namespaces" type="string[]" required>
  The list of namespace names.
</ResponseField>

<RequestExample>

```typescript Basic
const responseDelete = await index.listNamespaces();
// ["", "ns0", "ns1"]
```

</RequestExample>
