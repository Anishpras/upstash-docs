## Create a Collection

Create a Search Collection by navigating to the `Search` tab and clicking on the `Create Collection` button.

<Frame style={{ width: "600px" }}>
  <img src="/img/search/create-collection.png" />
</Frame>

A dialog with the following options will open:

- **Name:** Type a name for your collection (e.g. "product-search").

- **Region:** Choose the region for your index. For best performance, select the region closest to your application.

  _We plan to support additional regions and cloud providers. Feel free to send your requests to [support@upstash.com](mailto:support@upstash.com)._

Once you're done, click `Next`, choose a plan, and your Collection is ready:

<Frame style={{ width: "600px" }}>
  <img src="/img/search/collection-created.png" />
</Frame>

## Adding Data

Add searchable data to your Collection using our REST API, our SDKs, or directly in the dashboard.

### 1. Adding Data via Dashboard

Navigate to the `Data Browser` section of your Collection and click `Add document`.

<Frame style={{ width: "600px" }}>
  <img src="/img/search/add-data.png" />
</Frame>

A dialog with the following options will open:

<Frame style={{ width: "600px" }}>
  <img src="/img/search/add-document.png" />
</Frame>

- **Index:** A namespace for your data.

  _If you plan to query all documents in one place, you only need one index (e.g. "product-search"). If you plan to add multi-tenancy, so that each user can only search their own data, for example, you can create one index per user ("user-1", "user-2", etc.)._

- **ID:** An automatically generated ID.

- **Data:** The searchable data in string or stringified JSON format.

- **Fields:** Optional information attached to this document.
